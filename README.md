# 🍷 Cantina Vini

## SETUP IN 3 PASSI

### PASSO 1 — Supabase (crea le tabelle e carica i dati)
1. Vai su https://supabase.com/dashboard/project/gnnkpopqoxvgwuuiqdro/sql/new
2. Incolla TUTTO il contenuto del file `SETUP_SUPABASE.sql`
3. Clicca **RUN** — dovrebbe dire "Success"

### PASSO 2 — GitHub
1. Carica tutti questi file su GitHub (già fatto con GitHub Desktop)

### PASSO 3 — Vercel
1. Vai su https://vercel.com
2. Importa il repository GitHub
3. NON serve aggiungere variabili d'ambiente (sono già nel codice)
4. Clicca **Deploy**
5. Vercel ti dà un link tipo `cantina-vini.vercel.app`

### PASSO 4 — iPhone
1. Apri il link in Safari
2. Tocca **Condividi → Aggiungi a schermata Home**
3. L'app funziona anche offline in cantina!
